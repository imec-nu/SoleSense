# SoleSense

**SoleSense: A Shoe-Based Wearable for Sound Event Localization and Detection in Urban Environments**

This repository contains the implementation and hardware resources for **SoleSense**, a shoe-based wearable sensing platform that enables sound event detection and sound source localization using microphones embedded in the shoe sole.

## Repository Structure

This repository is organized into two major components.

### 1. SoleSense Design (`design/`)

This directory contains the hardware implementation of SoleSense, including:

- Hardware overview
- System architecture
- Hardware photos
- Firmware
- Hardware documentation

### 2. SoleFormer (`soleformer/`)

This directory contains the implementation of **SoleFormer**, a transformer-based model architecture used for sound event detection and sound source localization.

It includes:

- Model architecture
- Source code

---

## Project Overview

SoleSense explores shoe-based acoustic sensing by leveraging near-ground microphones for robust environmental sound perception.

The system supports:

- Sound Event Detection (SED)
- Sound Source Localization (SSL)
- SoleFormer transformer-based audio understanding
